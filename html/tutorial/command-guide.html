Command Guide

G'MIC itself furnishes canonical documentation, available through the G'MIC -help command (short form -h).

$gmic -help mix_rgb

gmic: GREYC's Magic for Image Computing.

Version 1.5.7.1, Copyright (C) 2008-2013, David Tschumperle.

(http://gmic.sourceforge.net)

-mix_rgb:

a11,a12,a13,a21,a22,a23,a31,a32,a33

Apply 3x3 specified matrix to RGB colors of selected images.

Default values: 'a11=1', 'a12=a13=a21=0', 'a22=1', 'a23=a31=a32=0' and 'a33=1'.

These online blurbs mirror the command entries in the G'MIC Handbook, less the example images. In a few words, these blurbs tell you exactly what you need to know. That may not be obvious at first.

Alas, the command documentation here still largely reflects the Handbook. As time permits, we'll augment the canonical stubs, furnish extended examples and applications. This will proceed no particular order; those that we use a good deal tend to migrate to the top of the priority stack.

The links to those augmented commands are boldface in the following table.
